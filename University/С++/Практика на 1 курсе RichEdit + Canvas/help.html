Задание 1

Описание:

Программа предназначена для поиска ключевого слова, которое введено с клавиатуры.

При поиске предоставляется возможность осуществлять его по вхождению/слово целиком  и учитывать регистр букв.

 

Работа с программой:

1.      Для запуска программы не требуется установка. Чтобы начать поиск, в окно ключевого поля нужно ввести слово для поиска. Оно не должно содержать пробелом и пунктуационных знаков. Программа полноценно масштабируема.

2.       - Главное меню программы.

В разделе «Файл» можно открыть/сохранить текст, выйти из программы. В разделе «Настройки» для первого задания можно выбрать «Не выделать предложение». Этот пункт означает что предложение содержащие искомый файл выделено не будет. В разделе «Инфо» можно открыть эту инструкцию.

Также в этом разделе можно выбрать скин программы.

3.      В поле ключевого поля уже присутствует слово «тест», а окно текста заполнено пробными предложениями.

4.      Объекты программы

Кнопка выполнения поиска, под кнопкой выводиться результат.

 

Эти флажки отвечают за параметры поиска.

Ключевое поле, из него береться слово для поиска.

5.      Найденное слово из ключевого поля выделает  в тексте красным цветом, само предложение с этим словом выделается толстым шрифтом.

Задание 2

Описание:

            Программа определяет пересечения прямой и окружности в декартовых координатах.

            По введенным данным строиться график.

           

Работа с программой:

1.      Для запуска программы не требуется установка. Чтобы выполнить задание нужно нажать кнопку «Рассичтать». Программа полноценно масштабируема.

2.       - Главное меню программы.

В разделе «Файл» можно открыть/сохранить текст, выйти из программы. В разделе «Настройки» для второго задания можно выбрать «Не очищать график» и «Не выводить координаты точек». Эти пункты означают что при рисовании график не будет очищен или не будут нарисованы координаты точек пресечения. В разделе «Инфо» можно открыть эту инструкцию.

Также в этом разделе можно выбрать скин программы.

3.      Координаты фигур и масштаб изначально уже заданы, их можно отредактировать.

4.      Объекты программы

Объект

Назначение

Кнопка выполнения расчетов и рисования графика.

Тут задаются параметры круга и его расположение

Тут задаются концы отрезка, с которым ищется пересечения окружности.

Тут указываются цвета рисования на графике.

В этом поле задается параметр рисования графика, (шаг в пикселях между единичным разделителем)

 

5.      Программа рассчитывает точки пересечения и по введенным данным строит график, и если такие точки есть, выделает эти точки, если выключена опция выводить координаты, то выводит координаты возле них.

 

6.      Сообщения программы

Статус

Сообщение

Причина

Ошибка
  • Не хватает данных
  • is not a valid floating point value

В какое-то из полей пустое или содержит недопустимое значение

Ошибка

Концы отрезка совпадают

Координаты точек концов отрезка равны

Ошибка

Радиус окружности не может быть отрицательным

В поле радиуса окружности введено не верное значение.

Ошибка

Для сохранения не хватает графика

Чтобы сохранить график как рисунок, он ещё не нарисован.

Информирование

Окружность проецируется в точку

Не критичная ошибка, если пересечение будет найдено, то эта точка будет центром окружности.

Информирование

Внимание! Возможно некоторые точки выходят за область графика

Круг или прямая выходят за область рисования на графике.
